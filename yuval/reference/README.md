# yuval/reference/

תיקייה זו מכילה **תמונות השראה לסגנון** של יובל (מעצב התמונות).

יובל סורק את הקבצים כאן בתחילת כל בקשת תמונה, מחלץ מהם סגנון, פלטת צבעים, קומפוזיציה ואלמנטים ויזואליים, ומשלב אותם ב-prompt ליצירת התמונה — כדי לשמור על **עקביות ויזואלית** בין כל התמונות בפרויקט.

הניחו כאן תמונות (PNG/JPG) שמייצגות את הקו העיצובי שאתם רוצים שיובל יחקה.
